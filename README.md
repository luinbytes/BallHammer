BallHammer is a [Darktide Mod Framework](https://github.com/Darktide-Mod-Framework/Darktide-Mod-Framework) mod with Perkaholic-style enemy overlays and configurable aim controls.

## Features

- Outlines and nameplates for priority enemies
- Dynamic boxes and counts for hordes
- Configurable aimbot, magnet triggerbot, and rage mode
- Distance, field-of-view, smoothing, and activation settings

## Requirements

- [Darktide Mod Loader](https://github.com/Darktide-Mod-Framework/Darktide-Mod-Loader)
- [Darktide Mod Framework](https://github.com/Darktide-Mod-Framework/Darktide-Mod-Framework)

## Installation

1. Install the Darktide Mod Loader and Darktide Mod Framework.
2. Copy this repository into the game's `mods` directory as `BallHammer`.
3. Add `BallHammer` to `mod_load_order.txt`.

Configure the mod from Darktide's mod options. The `/esp` command and the optional ESP keybind toggle the overlays during play.
